Demonstration: Integrating a Local LLM

Use Vibe mode to integrate a local LLM (Ollama) for paraphrase quality evaluation

Demonstration Overview

In this exercise, you'll use TRAE SOLO's Vibe mode to add an LLM evaluation panel to the paraphrase results page. This exercise demonstrates how Vibe mode can quickly integrate external APIs and display the results in your UI.

Why Vibe Mode?

Adding LLM evaluation is a focused task that Vibe mode handles well:

  • It involves a single API call to Ollama
  • The UI component is self-contained
  • Steering documents provide the React/TypeScript context
  • You can iterate quickly with follow-up prompts
A Note About Ollama

Ollama is already installed on the aiHackathon computers. It can understand both text and images you give it, and respond with text. Think of it as a local AI assistant that runs right on your machine, without needing an internet connection.

Step 1: Open the Paraphrasing Mentor Project

Open TRAE SOLO and import the existing Paraphrasing Mentor project from your Mac. Click Open Folder in the centre of the left panel, or click Select Project > Open Folder in the upper-left corner of the interface, then navigate to the Paraphrasing Mentor folder.

Once the project is open, ask the AI agent to help you run the app:

▶ Click to reveal the prompt
Help me run this app on localhost so I can see it in my browser.

Step 2: Write Your Prompt

Tell the AI agent what you want in plain English. Be clear about using the local LLM running on your machine:

▶ Click to reveal the prompt
Use the local LLM in Ollama (qwen 3.5) to evaluate the paraphrasing attempts.

Step 3: Test the Feature

Accept the changes and test them:

  1. Ensure Ollama is running (ollama serve)
  2. Submit a text for paraphrasing
  3. Verify the evaluation panel appears with a quality score
  4. Check that colour coding matches the score
  5. Stop Ollama and verify the error message displays correctly

Step 4: Iterate and Refine

Use follow-up prompts to improve the feature:

▶ Click to reveal example prompts
"Add a loading spinner while waiting for the LLM response"
"Make the evaluation panel collapsible with a toggle button"
"Add a history of previous evaluations for this text"
Key Takeaway

Vibe mode lets you add AI features to your app by simply describing what you want in plain English. The AI writes the code, you test it in your browser, and refine it with follow-up requests — no coding required.
