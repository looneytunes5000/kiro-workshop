AI-Driven Development with TRAE SOLO

Learn to leverage AI-native development to build features faster while maintaining code quality

In this workshop, you will learn how to leverage TRAE SOLO's AI-native development capabilities to build features faster while maintaining code quality and architectural consistency through hands-on exercises with a real serverless application.

Learning outcomes

By completing this workshop, you will gain hands-on experience with:

  • Vibe Coding for rapid feature implementation
  • Spec-Driven Development for complex, multi-file features
  • Steering to teach TRAE SOLO about your codebase and standards
  • Agent Hooks for automated workflows
  • MCP Integration to extend TRAE SOLO's capabilities
  • Cloud Agent for cloud-based execution environments

Duration

Estimated duration of this workshop is 3 to 4 hours.

Intended audience

The target audience for this workshop includes software development engineers, programmers, and developers who want to gain a deep understanding on using AI to deliver software development related outcomes.

Estimated cost

This workshop can be run locally or with cloud resources. Costs vary based on your setup. Remember to clean up any cloud resources after the workshop.
