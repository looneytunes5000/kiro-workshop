Bring Your Solution Back

If you want to keep working on your solution after the aiHackathon, you have two options to back up your project:

Option 1: Push to Your Personal GitHub Repository

You can use TRAE or any coding agent to push your entire project to a personal GitHub repository. Here is a sample prompt you can use:

▶ Click to reveal the sample prompt 
Push all files in this project directory to a new GitHub repository.
Steps:
1. Initialise a git repo if one doesn't already exist
2. Create a .gitignore file for this project type
3. Create a new repository on GitHub under my account
4. Commit all project files with a descriptive message
5. Push to the remote repository
6. Confirm the push was successful and provide the repository URL

Paste this prompt into TRAE's chat panel, and it will guide you through pushing your project. Make sure you are signed in to GitHub with your personal account before running the prompt.

Option 2: Upload to Your Personal OneDrive

Alternatively, you can upload your project directory to your personal OneDrive account. The OneDrive link is available on the MacBook at the venue.

Where to find the link

The OneDrive upload link is available on the MacBook at your workshop venue. Ask the organisers if you cannot locate it.
