TRAE SOLO Overview

Understand TRAE SOLO's core capabilities and AI-native workspace approach

What is TRAE SOLO?

TRAE SOLO is an AI-native workspace offering both web and desktop clients, featuring dual modes (MTC and Code). It provides a unified, efficient, and customizable AI collaboration experience covering scenarios from professional development to everyday office work.

Key Difference

Traditional IDEs: Code editor with AI bolted on as a plugin
TRAE SOLO: AI-native workspace where code, AI, and workflow are unified from the ground up

The TRAE SOLO Interface

TRAE SOLO's interface is designed for seamless AI collaboration. Here are the key components:

Editor Area

The main code editing area where you write and modify code. TRAE SOLO provides intelligent syntax highlighting, auto-completion, and inline AI suggestions as you type.

AI Chat Panel

The AI chat panel is your primary interface for conversational development. You can:

  • Ask questions about your codebase
  • Request code changes in natural language
  • Get explanations of complex code
  • Generate new features from descriptions
  • Debug issues with AI assistance

Agent Panel

The agent panel allows TRAE SOLO to work autonomously on larger tasks. You can assign multi-step tasks and TRAE SOLO will plan and execute them, showing you progress along the way.

Terminal

The integrated terminal lets you run commands, build projects, and test your code without leaving the workspace.

Task Progress Display

Track the progress of AI-driven tasks in real-time. See what TRAE SOLO is working on, review intermediate results, and intervene when needed.

Dual Clients

TRAE SOLO offers two client options to fit different workflows:

Web Client

Lightweight and accessible from any browser. Ideal for quick tasks, on-the-go development, or environments where installing software isn't possible. No setup required — just open and start working.

Desktop Client

Full-featured and independent of the web client. Offers deeper integration with your local environment, offline capabilities, and access to system-level tools. Best for professional development workflows.

Dual Modes

TRAE SOLO adapts to your skill level and needs through two distinct modes:

MTC Mode

Designed for non-developers. Focus on describing what you want in natural language while TRAE SOLO handles the implementation. Perfect for product managers, designers, and anyone who needs to build without writing code.

Code Mode

Built for developers who want AI assistance while maintaining full control over the code. Write, review, and refine code with AI as your pair programmer. Ideal for professional software development.

Cloud Agent

TRAE SOLO's Cloud Agent provides cloud-based runtime execution with unified management. The agent runs tasks in the cloud, enabling:

  • Consistent execution environment regardless of local setup
  • Centralized management of AI-driven tasks
  • Seamless switching between web and desktop clients
  • Scalable compute resources for complex operations

Input Types

TRAE SOLO supports multiple input methods to match how you work:

  • Text: Type your requests, descriptions, and questions directly
  • Voice: Speak your instructions for hands-free development
  • Attachments: Upload files, images, or documents for context
  • Skills: Leverage pre-built capabilities and custom extensions

Core Capabilities

TRAE SOLO offers five core capabilities that you'll explore throughout this workshop:

Vibe Coding

Rapid feature development through natural language. Describe what you want and TRAE SOLO helps you build it quickly while maintaining code quality and following your project's patterns.

Spec-Driven Development

For complex, multi-file features, use structured specifications to guide TRAE SOLO through planning and implementation. This ensures thorough consideration of requirements before writing code.

Steering

Teach TRAE SOLO about your codebase conventions, architectural patterns, and team standards through steering documents. This helps TRAE SOLO generate code that fits seamlessly into your project.

Agent Hooks

Automate repetitive workflows by configuring TRAE SOLO to respond to specific events or patterns. Agent Hooks let you customize TRAE SOLO's behavior to match your team's development practices.

MCP Integration

Extend TRAE SOLO's capabilities by connecting to external tools and services through the Model Context Protocol (MCP). This allows TRAE SOLO to interact with databases, APIs, version control, and more.

AI Models

TRAE SOLO integrates with advanced AI models to provide:

  • Code Generation: Generate code from natural language descriptions
  • Code Understanding: Explain complex code and architecture
  • Refactoring: Suggest and implement code improvements
  • Debugging: Identify and fix bugs with AI assistance
  • Testing: Generate test cases and validate code

Typical Workflow

A typical development workflow with TRAE SOLO looks like this:

  1. Describe your goal: Tell TRAE SOLO what you want to build or change
  2. Review the plan: TRAE SOLO may propose a plan for complex changes
  3. Implement: TRAE SOLO generates the code changes
  4. Review and refine: Review the changes, ask for modifications if needed
  5. Test: Run tests to verify the changes work correctly
Important Note

While TRAE SOLO is powerful, you remain in control. Always review AI-generated code before accepting changes. TRAE SOLO is a collaborator, not a replacement for your judgment and expertise.
