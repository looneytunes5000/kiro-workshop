Installing TRAE SOLO Desktop

In this step, you'll download and install the TRAE SOLO Desktop client on your machine. TRAE SOLO is available for Windows, macOS, and Linux.

System Requirements
  • Windows: Windows 10 or later (64-bit)
  • macOS: macOS 11.0 (Big Sur) or later
  • Linux: Ubuntu 18.04+, Debian 10+, or equivalent
  • RAM: 8GB minimum, 16GB recommended
  • Disk: 500MB for installation

Download TRAE SOLO Desktop

  1. Visit the TRAE SOLO download page

    Navigate to the official TRAE SOLO download page and select the installer for your operating system.

  2. Download the installer

    Click the download button for your platform. The installer package is approximately 200-300MB.

  3. Run the installer

    Once the download completes, run the installer and follow the on-screen instructions.

    • Windows: Run the .exe installer
    • macOS: Open the .dmg file and drag TRAE SOLO to Applications
    • Linux: Use the provided .deb or .rpm package
  4. Launch TRAE SOLO

    After installation completes, launch TRAE SOLO from your applications menu or desktop shortcut.

First Launch

When you first launch TRAE SOLO Desktop, you'll see a welcome screen with several options:

  • Open a folder - Open an existing project directory
  • Clone a repository - Clone a Git repository directly
  • Create a new project - Start a fresh project from a template

For this workshop, you'll be cloning the provided workshop repository. Don't worry about this yet - we'll cover it in the next steps after you've completed the TRAE SOLO setup and authentication.

Verify Installation

To verify that TRAE SOLO is installed correctly:

  1. Open TRAE SOLO Desktop
  2. Check that the interface loads without errors
  3. Look for the TRAE SOLO AI panel in the sidebar - this confirms the AI features are available
  4. Open the command palette (Ctrl+Shift+P or Cmd+Shift+P) and type "TRAE" - you should see TRAE SOLO-specific commands

Troubleshooting

If you encounter issues during installation:

  • Installation fails on Windows: Make sure you have administrator privileges and try running the installer as administrator
  • TRAE SOLO won't launch on macOS: Check System Preferences > Security & Privacy and allow TRAE SOLO if it's blocked
  • AI features not available: Ensure you have an active internet connection, as TRAE SOLO's AI features require cloud connectivity
  • Slow performance: Close other resource-intensive applications and ensure you have at least 8GB of RAM available
