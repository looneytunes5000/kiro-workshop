Getting Started with TRAE SOLO

TRAE SOLO offers two ways to access its AI-powered development environment. Choose the option that best fits your needs and workflow.

Quick Recommendation

For this workshop, we recommend starting with the web client for instant access. You can always switch to the desktop client later for long-term development.

Option 1: Web Client (Recommended for Quick Start)

The web client provides immediate access to TRAE SOLO without any installation. Simply open your browser and start building.

  • No installation needed - Access TRAE SOLO directly from your browser
  • Internet connection required - Works anywhere with a stable connection
  • Ready-to-use cloud environment - Pre-configured and ready to go
  • Great for temporary needs - Perfect for workshops, quick tasks, and trying things out
  • Rapid validation - Test ideas and prototypes without setup overhead

Option 2: Desktop Client

The desktop client provides a full-featured development experience with additional capabilities and offline access to your editor.

  • Download and install - Available for Windows, macOS, and Linux
  • Independent architecture - Runs natively on your machine for optimal performance
  • Supports diverse input types - Text, voice, attachments, and skills
  • Real-time progress tracking - Monitor AI agent tasks directly in the interface
  • Direct preview in chat - See results and previews without leaving the IDE

Choosing Your Client

When to Use Each
  • Web Client: Best for quick tasks, workshops, temporary environments, and when you need instant access without installation
  • Desktop Client: Best for long-term development, daily work, projects requiring diverse input types, and when you want the full feature set

Both clients share the same core AI capabilities and can access your projects. Your choice depends on your immediate needs and workflow preferences.

Next Steps

After choosing your client:

  1. If you chose the desktop client, follow the Desktop Client page for installation instructions
  2. Then proceed to Setup Account to create or sign in to your TRAE SOLO account
