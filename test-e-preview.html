AI-Driven Development with TRAE SOLO

In this workshop, you will learn how to leverage TRAE SOLO's AI-native development capabilities to build features faster while maintaining code quality and architectural consistency through hands-on exercises with a real serverless application.

Learning outcomes

By completing this workshop, you will gain hands-on experience with:

→ Vibe Coding — Rapid feature implementation using AI-assisted development workflows
→ Spec-Driven Development — Complex, multi-file features guided by detailed specifications
→ Steering — Teach TRAE SOLO about your codebase and development standards
→ Agent Hooks — Automated workflows that respond to code changes
→ MCP Integration — Extend TRAE SOLO's capabilities with custom tool integrations
→ Cloud Agent — Run AI agents in the cloud for scalable, persistent development workflows

Duration

Estimated duration of this workshop is 3 to 4 hours.

Intended audience

The target audience for this workshop includes software development engineers, programmers, and developers who want to gain a deep understanding on using AI to deliver software development related outcomes.

Workshop costs

If you are running this workshop at a hosted event, you will not incur costs.

If you are running this event on your own account, estimated costs are minimal for the duration of the workshop. Remember to clean up resources after the workshop to avoid ongoing charges.
