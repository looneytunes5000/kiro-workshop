git-commit

git-commit

Development Tools

This skill writes commit messages based on code changes and stages the correct files. It follows the Conventional Commits format.

Example: Commit changes without writing a message

After editing files, tell TRAE to commit. The skill reads the changes and generates a message like feat(auth): add login form validation.

Example: Keep commit history consistent

The skill picks the right type (fix, feat, chore, refactor) and scopes changes so the project history stays organised.

Example: Split unrelated changes into separate commits

If you edited both a button component and a database query, the skill separates them into two commits so each one stays focused on a single change.

react-best-practices

react-best-practices

Development Tools

This skill reviews React and Next.js code for common issues. It identifies re-rendering problems, missing hook dependencies, and structural patterns.

Example: Identify rendering issues

The skill spots unnecessary re-renders, missing dependencies in hooks, and components that could be split. It explains what to change and why.

Example: Refactor growing codebases

As projects expand, code that works can become hard to maintain. The skill suggests extracting logic into custom hooks or memoising expensive calculations.

Example: Learn patterns through feedback

Each suggestion includes an explanation of why a pattern is preferred, helping you build familiarity with React conventions over time.

webapp-testing

webapp-testing

Development Tools

This skill creates browser tests using Playwright. It can click buttons, fill in forms, and confirm that expected content appears on screen.

Example: Test a sign-up form

The skill enters a name, email, and password, clicks submit, and checks that the confirmation message appears. If something breaks, the test reports the failure.

Example: Check pages after making changes

After updating code, the skill runs through key pages and confirms navigation links work, images load, and interactive elements respond.

Example: Capture screenshots during tests

The tool takes screenshots at key steps, providing visual records that help with debugging when a test fails.

composition-patterns

composition-patterns

Development Tools

This skill restructures complex React components into smaller, reusable pieces. It identifies components that have too many responsibilities and suggests ways to split them.

Example: Split a large component

When a component handles multiple responsibilities — rendering a form, validating input, and submitting data — the skill breaks it into focused sub-components so each piece has a single responsibility.

Example: Create reusable components

The skill helps design components that adapt to different content across the project, reducing duplication of similar UI patterns like card layouts.

Example: Replace boolean props with composition patterns

If a component has grown to accept multiple boolean props like isLarge, showDetails, the skill suggests using compound components or render props instead.

agent-browser

agent-browser

Productivity

This skill lets TRAE control a web browser. It can open websites, fill out forms, click buttons, scroll pages, and extract data.

Example: Extract data from a website

Tell the skill to visit a page and collect information like prices, titles, or links. It navigates the site and returns the content you need.

Example: Fill out the same form with different data

The skill types into fields, selects dropdowns, clicks submit, and waits for the response — repeating the process for each set of data.

Example: Check how a page looks

The skill can take a screenshot of any page. It can also click through navigation menus to verify that pages load, which is useful during development and testing.

brainstorming

brainstorming

Productivity

This skill helps organise your ideas before you start coding. You share a rough concept, and it asks clarifying questions and produces a structured plan.

Example: Clarify a rough idea

Say you want to build a dashboard but are not sure where to start. The skill asks questions about data sources, layout, and users, and produces a step-by-step plan.

Example: Compare different approaches

The skill helps weigh options like comparing two databases or two UI layouts. You get a balanced view of trade-offs before building.

Example: Write a specification document

For features that involve multiple components and data flows, the skill helps write a clear specification that other skills can follow during implementation.

figma

figma

UI Design

This skill reads a Figma design file and generates frontend code from it. It extracts layout, colours, typography, and spacing from the design.

Example: Convert a Figma mockup into code

Pass a Figma file link to the skill. It reads the design layers, extracts styles, and generates React components that match the visual layout.

Example: Extract design tokens

The skill pulls out colours, spacing values, font sizes, and other design tokens from your Figma file. You get a consistent set of variables for your project.

Example: Flag Figma setup issues before coding

When a Figma file has unnamed layers, inconsistent spacing, or missing component structure, the skill flags the problems so you can decide how to handle them.

frontend-design

frontend-design

UI Design

This skill builds interfaces with a distinct visual style from scratch. It handles layout, colour scheme, typography, and motion effects.

Example: Build a landing page from a description

Describe the sections you need — hero, features, pricing, testimonials — and the skill generates a styled page with appropriate layouts and visual hierarchy.

Example: Improve an existing page's appearance

The skill reviews the layout and suggests adjustments — rearranging elements, adjusting spacing, adding visual accents — that improve the design without changing the content.

Example: Build responsive components

The skill creates components that adapt to different screen sizes. Your layout works across mobile, tablet, and desktop.

frontend-skill

frontend-skill

UI Design

This skill creates clean interfaces with restrained style and clear visual hierarchy. It focuses on making important content stand out while avoiding visual clutter.

Example: Build a simple app interface

For an app shell with a sidebar, header, and content area, the skill creates a clean structure with proper spacing and clear visual hierarchy.

Example: Tighten an uneven UI

If your interface has inconsistent paddings, misaligned buttons, or unclear content priority, the skill makes targeted adjustments to spacing, alignment, and visual emphasis.

Example: Add subtle motion

The skill adds transitions, hover effects, and micro-interactions where appropriate. Just enough motion to make the interface feel responsive without being distracting.

chart-visualization

chart-visualization

Data & Content

This skill examines your data and selects the most suitable chart type — bar, line, pie, scatter, and others from the available options.

Example: Show trends over time

Provide monthly numbers and the skill automatically selects a line chart, which shows trends clearly. The visual makes it easy to see whether values are rising, falling, or flat.

Example: Compare items side by side

For comparing product sales across regions, the skill picks a grouped bar chart. You see which product is leading in which region at a glance.

Example: Find patterns or outliers

When exploring data without a specific question, the skill recommends charts that reveal patterns. A scatter plot might show a correlation, or a histogram might highlight unusual outliers.

data-analysis

data-analysis

Data & Content

This skill reads Excel or CSV files and runs SQL queries against them. You can ask for totals, averages, filtered lists, or multi-table reports.

Example: Summarise a spreadsheet

Upload a spreadsheet and ask something like "What is the total revenue per product?" The skill writes the query and returns the summarised results.

Example: Join data from multiple sheets

If data is spread across multiple tabs — customers in one tab, orders in another — the skill connects them so you can ask questions involving both.

Example: Filter and sort data

Ask the skill to find records matching criteria — like all orders over $500 from last month, sorted by date. It applies the filters and returns the relevant rows.

canvas-design

canvas-design

Data & Content

This skill creates original visual content like posters and covers in PNG or PDF format. It handles composition, typography, and colour choices.

Example: Create an event poster

Describe your event — title, date, venue — and the style you want (minimal, bold, retro, modern). The skill generates a poster with balanced composition and appropriate typography.

Example: Design social media graphics

The skill creates compositions sized correctly for each platform — blog covers, LinkedIn banners, or visual quotes.

Example: Generate print-ready materials

The skill can produce PDF output for report covers, booklet front pages, or certificates, using typography and layout standards suitable for printing.

doc-coauthoring

doc-coauthoring

Data & Content

This skill helps write and refine documents through an iterative process. You provide ideas and context, and it structures them into clear documents.

Example: Write a technical specification

Share rough notes about a feature. The skill organises them into sections — overview, requirements, architecture, timeline — and refines each section iteratively.

Example: Create a project proposal

The skill guides you through writing a proposal piece by piece, asking for context and helping structure arguments so the final document clearly communicates scope and resource needs.

Example: Review an existing document

If you already have a draft, the skill reviews it for clarity, structure, and completeness. It identifies gaps, suggests reorganisations, and helps polish the language.
