Web vs Desktop Clients

Choose the right TRAE SOLO client for your workflow

Overview

TRAE SOLO provides both web and desktop clients, delivering comprehensive upgrades in interaction modes, operating architecture, and feature experience. Each client is designed to serve different workflows and use cases.

Understanding the strengths of each client helps you choose the right tool for the task at hand.

Web Client

The web client offers a lightweight, accessible experience that works anywhere with an internet connection.

  • Lightweight and convenient - Available anytime, anywhere with internet access
  • No download or installation required - Ready-to-use cloud environment, just open your browser
  • Simple task management - Efficient interface for initiating and tracking tasks
  • Lightweight interactions - Does not consume local system resources
Best For

Temporary needs, working on the go, rapid validation, and situations where you need quick access without installation.

Desktop Client

The desktop client is an independent architecture designed specifically for AI agents, offering deep performance optimization for coding workflows.

  • Independent architecture - Designed specifically for AI agents, does not rely on TRAE IDE for operation
  • Diverse input types - Supports text, voice, attachments, and skills
  • Real-time progress display - Shows current progress of tasks as they execute
  • Automatic summarization - Output content is automatically summarized for quick review
  • Direct preview and acceptance - Preview and accept results directly within the chat interface
  • Deep performance optimization - Optimized specifically for coding agent workflows
Best For

Long-term development, complex projects, and workflows that benefit from deep IDE integration and advanced AI coding features.

Comparison

Feature Web Client Desktop Client
Installation Required No Yes
Access Anywhere Yes Limited to installed device
Local Resource Usage Minimal Moderate
Voice Input Limited Full Support
Real-time Task Progress Basic Advanced
Automatic Summarization Basic Full Support
Direct Result Preview Limited Full Support
Performance Optimization Standard Deep Optimization
Ideal Use Case Quick tasks, on-the-go Long-term, complex projects

Switching Between Clients

Your work syncs seamlessly across both clients. You can start a task on the web client while on the go, then continue working on it in the desktop client when you return to your desk.

Pro Tip

Start on web for quick validation and ideation, then switch to desktop for deep development work. Your projects, conversations, and context follow you across clients.
