Take Kiro Home (Local Installation)

Set up Kiro on your personal machine for continued use after the workshop

Why Install Locally?

While the workshop environment is great for learning, installing Kiro locally gives you:

  • Full control over your development environment
  • Access to your own projects and repositories
  • Ability to customize settings and extensions
  • Offline access to the editor (AI features still require internet)
  • Integration with your existing tools and workflows
Workshop Benefit

One of the key takeaways from this workshop is having Kiro installed and configured on your machine, ready to use in your daily development work.

Installation Options

Option 1: Standard Installation

Download and install Kiro directly from the official website:

  1. Visit the Kiro download page
  2. Select your operating system
  3. Download and run the installer
  4. Follow the installation wizard

Option 2: Package Manager

For developers who prefer package managers:

# macOS with Homebrew brew install --cask kiro # Windows with winget winget install Amazon.Kiro # Linux with snap sudo snap install kiro

Post-Installation Setup

After installing Kiro:

  1. Sign in with AWS Builder ID - This enables AI features
  2. Configure your preferences - Theme, font size, keybindings
  3. Install extensions - Language support, themes, and tools
  4. Open a project - Start with your own or the workshop demo app

Migrating from Workshop Environment

If you want to continue working with the workshop demo application:

  1. Clone the demo repository to your local machine
  2. Open the project in your local Kiro installation
  3. Run npm install to install dependencies
  4. Start the development server with npm start
Tip

Copy any steering documents or configurations you created during the workshop to your local installation. This preserves your learning and setup.

Recommended Extensions

Consider installing these extensions to enhance your Kiro experience:

  • Language Support: TypeScript, Python, Java, etc.
  • Linting & Formatting: ESLint, Prettier, Black
  • Testing: Test runners for your language
  • Git Integration: GitLens or similar
  • Themes: Choose a theme that works for you

Getting Help

If you encounter issues:

  • Check the Kiro documentation
  • Visit the community forum
  • Review the troubleshooting guide
  • Contact support through the IDE
