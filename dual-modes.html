MTC vs Code Modes

TRAE SOLO offers two modes designed for different user groups

Overview

To accommodate users from different backgrounds, TRAE SOLO offers two distinct modes: MTC (More Than Coding) and Code. Each mode is tailored to a specific audience and set of tasks, ensuring that every user gets the most effective experience for their needs.

MTC Mode (More Than Coding)

MTC mode is designed for users who are not involved in software development but still want to leverage AI-powered assistance for their daily work.

  • Target users - Product managers, data analysts, operations professionals, and other non-developers
  • Ideal for - Handling documents, data analysis, presentations, and similar materials
  • No coding knowledge required - Interact entirely through natural language
  • Natural language interaction - Describe what you need in plain language, and the AI handles the rest
Who Should Use MTC Mode

If your daily work involves documents, spreadsheets, presentations, data analysis, or project management rather than writing code, MTC mode is designed for you.

Code Mode

Code mode is designed for developers who are accustomed to agent-driven development workflows and need full IDE capabilities.

  • Focus areas - Coding, debugging, repository management, and Git workflows
  • Ideal for - Software development, feature implementation, code review, and refactoring
  • Full IDE capabilities - Access to code navigation, IntelliSense, debugging tools, and version control
  • Advanced AI coding features - Spec-driven development, vibe coding, agent hooks, and more
Who Should Use Code Mode

If you write code, manage repositories, review pull requests, or build software, Code mode provides the full power of TRAE SOLO's AI development capabilities.

Choosing Your Mode

Use this guide to decide which mode fits your role and task type:

Factor MTC Mode Code Mode
Your Role Product, Data, Operations, Marketing Developer, Engineer, Technical Lead
Primary Tasks Documents, Data, Presentations Coding, Debugging, Code Review
Coding Knowledge Not Required Expected
Interaction Style Natural Language Only Natural Language + Code
Output Type Documents, Analysis, Content Code, Features, Fixes
IDE Features Needed No Yes

Switching Modes

TRAE SOLO makes it easy to switch between modes based on your current task. You are not locked into a single mode - switch as your needs change throughout the day.

When to Switch

Switch to MTC mode when you need to work on documents, analyze data, or create presentations. Switch to Code mode when you need to write code, debug issues, or manage repositories. Many users switch between modes multiple times per day as their tasks change.
