Getting Started

Welcome to the hands-on portion of the TRAE SOLO workshop! In this section, you'll set up your development environment and prepare to start building with TRAE SOLO.

What You'll Do

In this section, you will:

  • Access TRAE SOLO (web or desktop)
  • Setup your account
  • Clone the workshop repository

Before You Begin

Make sure you have the following ready:

  • A computer with administrator access
  • Stable internet connection
  • TRAE SOLO account (you'll set this up in this section)
  • Basic familiarity with your operating system's terminal/command prompt
Important Note

The workshop provides a pre-configured application repository. You don't need to create a new project from scratch - you'll be working with an existing codebase that demonstrates real-world patterns.

Next Steps

Follow the pages in this section in order:

  1. Get Started - Choose between web client (instant access) or desktop client (full features)
  2. Desktop Client - Download and install the desktop client if you choose that route
  3. Setup Account - Create or sign in to your TRAE SOLO account to access all features

Once you've completed these steps, you'll be ready to dive into the core concepts and hands-on exercises!
