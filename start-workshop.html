Getting Started

Welcome to the hands-on portion of the Kiro workshop! In this section, you'll set up your development environment and prepare to start building with Kiro.

What You'll Do

In this section, you will:

  • Install the Kiro IDE on your machine
  • Configure Kiro for local development
  • Authenticate with AWS Builder ID
  • Clone the workshop application repository

Before You Begin

Make sure you have the following ready:

  • A computer with administrator access
  • Stable internet connection
  • AWS Builder ID credentials (you'll set this up in this section)
  • Basic familiarity with your operating system's terminal/command prompt
Important Note

The workshop provides a pre-configured application repository. You don't need to create a new project from scratch - you'll be working with an existing codebase that demonstrates real-world patterns.

Next Steps

Follow the pages in this section in order:

  1. Install Kiro - Download and install the Kiro IDE on your machine
  2. Take Kiro Home - Configure Kiro for local development with your preferences
  3. Authenticate with Builder ID - Set up AWS Builder ID authentication to access workshop resources

Once you've completed these steps, you'll be ready to dive into the core concepts and hands-on exercises!
