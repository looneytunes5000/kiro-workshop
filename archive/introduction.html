Introduction

Welcome to the TRAE SOLO Hands-on Workshop

Overview

Welcome to the VTC aiHackathon! In this workshop, you'll explore TRAE SOLO's AI-native development capabilities through practical, hands-on exercises with a real serverless application.

TRAE SOLO is an AI-native workspace offering both web and desktop clients, featuring dual modes (MTC and Code) designed for different user groups. It provides a unified, efficient, and customisable AI collaboration experience covering scenarios from professional development to everyday office work.

Multitasking

TRAE SOLO enables you to manage multiple tasks simultaneously within a single project, breaking the limitations of traditional serial task execution.

Chat + Development

Freely switch between different tasks - focus on module development while asking additional questions. This prevents both your and AI's thinking flow from being confined to a single chat.

Parallel Tasks

Create multiple independent tasks within the same project, each dedicated to different functional modules (such as frontend and backend), enabling parallel advancement of multiple tasks.

Dual Modes

TRAE SOLO features two distinct modes designed for different user groups and tasks:

MTC Mode

Designed for non-developers including product managers, analysts, and operations staff. MTC Mode excels at creating documents, analysing data, building presentations, and handling everyday office productivity tasks through natural language interaction.

Code Mode

Built for developers who need coding, debugging, repository management, and Git workflow capabilities. Code Mode provides deep codebase understanding, intelligent suggestions, and seamless integration with your development tools and processes.

Cloud Agent

TRAE SOLO includes a Cloud Agent - a cloud-based runtime environment that provides:

  • Unified dependency management - No need to manually install or configure packages
  • Isolated execution - Each session runs in its own clean environment
  • No local setup required - Start coding immediately without configuring your machine

The Cloud Agent ensures consistent, reproducible environments and eliminates the "works on my machine" problem.

Demo Application

Throughout this workshop, you'll work with a paraphrasing mentoring application for students. This application provides a realistic codebase to practice TRAE SOLO's capabilities on, including:

  • Frontend - A web interface for creating and viewing posts
  • Backend APIs - Serverless functions handling CRUD operations
  • Database - Persistent storage for posts and user data
  • Authentication - User identity and access management

The application is designed to be simple enough to understand quickly, yet complex enough to demonstrate real-world development scenarios.

TRAE SOLO's Core Capabilities

Throughout this workshop, you'll explore six key capabilities that make TRAE SOLO unique:

Vibe Coding

Rapid feature development through natural language descriptions. Describe what you want in plain English, and TRAE SOLO helps you build it quickly while maintaining code quality.

Spec-Driven Development

For complex, multi-file features, use structured specifications to guide TRAE SOLO through planning and implementation. This ensures thorough consideration of requirements, edge cases, and architectural impact before writing code.

Steering

Teach TRAE SOLO about your codebase conventions, architectural patterns, and team standards. Steering documents help TRAE SOLO understand your project's unique context and generate code that fits seamlessly.

Agent Hooks

Automate repetitive workflows by configuring TRAE SOLO to respond to specific events or patterns. Agent Hooks let you customise TRAE SOLO's behaviour to match your team's development practices.

MCP Integration

Extend TRAE SOLO's capabilities by connecting to external tools and services through the Model Context Protocol (MCP). This allows TRAE SOLO to interact with databases, APIs, version control, and more.

Cloud Agent

A cloud-based runtime environment with unified dependency management and isolated execution. The Cloud Agent eliminates local setup requirements and ensures consistent, reproducible development environments.

Workshop Structure

The workshop is organised into modules that build on each other:

→ Getting Started (~30 minutes)

Pacing Yourself

This workshop is designed to be completed in 3-4 hours. However, you can adjust the pace based on your experience level and interests:

  • Beginners - Take your time with each exercise, read all the explanations, and don't hesitate to experiment
  • Experienced developers - You may move through familiar concepts more quickly and focus on advanced topics
  • Time-constrained - Focus on the core exercises (Steering, Vibe Coding, Spec-Driven Development) and skip optional sections

Workshop Conventions

Throughout this workshop, you'll encounter:

  • Code blocks - Shown in monospace font with syntax highlighting
  • Step-by-step instructions - Numbered lists for sequential actions
  • Tips and notes - Additional context and best practices
  • Challenge exercises - Optional tasks to deepen your understanding
