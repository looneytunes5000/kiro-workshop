Authenticate with Builder ID

Set up AWS Builder ID authentication to enable Kiro's AI features

Setting Up AWS Builder ID Authentication

AWS Builder ID is a free, personal identity that allows you to explore AWS services and use AWS-powered tools like Kiro. In this step, you'll authenticate Kiro with your AWS Builder ID to enable AI features and access workshop resources.

What is AWS Builder ID?

AWS Builder ID is a personal AWS identity that provides free access to select AWS services and tools. It's separate from AWS IAM Identity Center and doesn't require an AWS account. You can use it to:

  • Access AWS Workshop Studio
  • Use AI-powered development tools
  • Explore AWS services in a sandbox environment

Create or Sign In with Builder ID

  1. Open Kiro IDE

    Launch Kiro if it's not already open.

  2. Access the Authentication Prompt

    When you first use Kiro's AI features, you'll be prompted to sign in. If you don't see a prompt:

    • Look for a "Sign In" button in the Kiro AI panel
    • Or go to Settings > AWS > Authentication
  3. Choose Builder ID

    Select "AWS Builder ID" as your authentication method (not AWS IAM Identity Center).

  4. Complete the Authentication Flow

    A browser window will open asking you to:

    • Sign in if you already have a Builder ID
    • Create a new Builder ID if you don't have one
  5. Authorize Kiro

    After signing in, you'll be asked to authorize Kiro to access your Builder ID. Click "Allow" or "Authorize".

  6. Return to Kiro

    Once authorized, you'll be redirected back to Kiro. The AI panel should now show that you're authenticated.

Creating a New Builder ID

If you don't have a Builder ID, you'll need to create one:

  1. Enter your email address

    Use a personal email address (not a work email, as Builder ID is for personal use).

  2. Verify your email

    Check your email for a verification code from AWS and enter it on the verification page.

  3. Create a password

    Choose a strong password for your Builder ID account.

  4. Complete your profile

    Enter your name and any other required information.

Important Notes
  • Builder ID is free: There is no cost to create or use a Builder ID
  • Personal use only: Builder ID is intended for personal, non-commercial use
  • Not an AWS account: Builder ID is separate from an AWS account and doesn't provide access to the AWS Management Console
  • Workshop access: Your Builder ID gives you access to this workshop and its resources

Verify Authentication

To verify that authentication is working correctly:

  1. Open the Kiro AI chat panel
  2. Type a simple question like "Hello, are you working?"
  3. Verify that you receive a response from the AI
  4. Check the status indicator in the AI panel - it should show "Connected" or similar

Troubleshooting

If you encounter authentication issues:

  • Browser doesn't open: Check your default browser settings and ensure pop-ups aren't blocked
  • Authorization fails: Make sure you're using Builder ID, not IAM Identity Center
  • AI features still unavailable: Try restarting Kiro after authentication
  • Session expires: You may need to re-authenticate periodically - simply repeat the sign-in process
  • Network issues: Ensure you have a stable internet connection, as AI features require cloud connectivity

You're Ready!

Congratulations! You've completed the setup phase. You now have:

  • Kiro IDE installed and configured
  • AWS Builder ID authentication set up
  • AI features enabled and working
