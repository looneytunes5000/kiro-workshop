GenAI Portal

Web-based gateway for LLM, image and video generation models

What is the GenAI Portal?

The GenAI Portal is a web-based platform that provides contestants with access to a range of advanced generative AI models through a unified interface. Instead of managing multiple accounts and tools, you can use one portal to generate text, images, and videos — all optimised for hackathon projects.

Available Capabilities

LLM Models

Access large language models for text-based tasks:

  • Text Generation — Create article drafts, marketing copy, documentation, and email templates
  • Code Completion — Generate boilerplate code, API endpoints, and utility functions
  • Summarization — Condense long documents, reports, or meeting notes into key points
  • Analysis — Review data, extract insights, and generate structured outputs from unstructured input

Image Generation

Create visual assets from text descriptions:

  • Text-to-Image — Generate images from detailed text prompts describing style, content, and composition
  • Style Variations — Explore multiple artistic styles (photorealistic, illustration, 3D render, etc.)
  • Image Editing — Modify existing images with in-painting, out-painting, or style transfers

Video Generation

Produce short video content:

  • Text-to-Video — Generate short video clips from descriptive prompts
  • Image-to-Video — Animate static images with motion effects
  • Avatar Videos — Create talking-head presentations with custom scripts
Tip

Start with the LLM models for most hackathon tasks. Use image and video generation when your project specifically needs visual content — they're powerful but can take longer to generate.

How Contestants Can Use It

Here are practical ways to integrate the GenAI Portal into your hackathon projects:

  • Content Creation: Generate blog posts, social media copy, or product descriptions for your project's marketing materials
  • UI Assets: Create placeholder images, icons, or illustrations for your frontend design while you build
  • Demo Videos: Produce short video content for your project pitch or product demo
  • Code Assistance: Use LLM models to generate test data, API schemas, or documentation drafts

Workflow Integration

  1. Log in to the GenAI Portal with your provided credentials
  2. Select the model type you need (LLM, Image, or Video)
  3. Enter your prompt or upload your input
  4. Review the generated output and refine your prompt if needed
  5. Download or copy the output for use in your project
