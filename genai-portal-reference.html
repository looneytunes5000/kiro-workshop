GenAI Portal Reference

Quick reference guide for the GenAI Portal

Accessing the Portal

  1. Navigate to the GenAI Portal URL provided to contestants
  2. Sign in with your hackathon account credentials
  3. Dashboard shows available model categories: LLM, Image, and Video
  4. Click on a category to access the model interface
Tip

Bookmark the portal URL for quick access during the hackathon. Session timeouts may occur, so keep your login details handy.

LLM Usage Quick Guide

Prompt Best Practices

  • Be specific: Include details about format, tone, length, and audience
  • Provide context: Share background information so the model generates relevant output
  • Use examples: Show a sample of the output style you want
  • Iterate: Refine prompts based on results — small changes often produce better output
Good prompt:
"Write a 3-paragraph product description for a task management app. Target audience: startup founders. Tone: professional but friendly. Include key features: drag-and-drop boards, team chat, deadline reminders."

Common Use Cases

Use Case Prompt Tip
Documentation Paste code snippets and ask for comments/explanations
Test Data Request JSON/CSV with specific fields and sample values
API Schemas Describe your data model and ask for OpenAPI/Swagger format
Summarization Paste long text and ask for key points in bullet format
Translation Specify source and target languages clearly

Image Generation Guide

Prompt Format

Structure your image prompts with these elements:

  • Subject: What the image should depict
  • Style: Photorealistic, illustration, 3D render, watercolour, etc.
  • Composition: Close-up, wide shot, portrait, landscape
  • Lighting: Natural light, studio lighting, dramatic shadows
  • Color palette: Warm tones, monochrome, pastel, vibrant
Example:
"A modern mobile app dashboard for task management, flat illustration style, clean white background with blue and green accents, showing boards, team avatars, and progress bars — minimalist design, professional look"

Resolution Settings

  • Standard: 512x512 — fastest generation, good for previews
  • High: 1024x1024 — recommended for project assets
  • Ultra: 2048x2048 — best quality, longer generation time

Video Generation Guide

Input Options

  • Text Prompt: Describe the scene, setting, and motion you want
  • Image Input: Upload a still image to animate or extend
  • Script + Avatar: Provide text and select a talking-head avatar

Limits and Formats

  • Duration: Maximum 15 seconds per generation
  • Resolution: Up to 1080p (1920x1080)
  • Output formats: MP4 (H.264) supported by most platforms
  • Generation time: 2-5 minutes depending on length and quality
Important

Video generation consumes more compute resources. Generate early in the hackathon to avoid queues during final presentation prep.

Troubleshooting Tips

Issue Solution
Generation is too slow Try a lower resolution or simpler prompt
Output quality is poor Add more detail to your prompt; specify style and composition
Session expired Refresh the page and log in again; your work is not saved automatically
Image looks generic Specify unique stylistic elements, lighting, or perspective
Video has artifacts Reduce complexity; avoid fast or extreme motion in prompts
Quick Reminder

Save or download all generated content immediately. The portal may not retain your outputs between sessions.
