VTC GenAI Portal Browser Automation

Automating VTC GenAI Portal interactions with Playwright

When Do You Need Page Automation?

If your hackathon solution involves automatically generating images or videos from the VTC GenAI Portal — for example, a pipeline that produces visual assets on demand, or a bot that creates demo content — you may want to automate the portal with Playwright. This page explains the concepts so you can prompt TRAE to write the automation scripts for you.

If your project does not need automated image or video generation, you can skip this page entirely.

What is Playwright?

Playwright is a browser automation library that can control a browser programmatically — clicking elements, filling forms, waiting for content, and downloading files, all without human interaction. Playwright is one of the MCP tools available and pre-installed in TRAE, so you can start using it immediately without any setup. Learn more about MCP tools in TRAE.

Important

Playwright is pre-installed and available in TRAE. You don't need to install anything. Just ask TRAE to write a Playwright script and it will be ready to run.

Automating the VTC GenAI Portal

The VTC GenAI Portal is a web application with text-to-image and text-to-video features. Automating it with Playwright typically involves:

  • Navigating to https://genai.vtc.edu.hk/ and logging in
  • Selecting the model type (image or video generation)
  • Filling in the prompt text and any settings (resolution, format, aspect ratio)
  • Waiting for the model to finish generating content
  • Downloading the generated file to your project directory

Example Prompts for TRAE

Here are examples of prompts you can use with TRAE to generate automation scripts:

▶ Image Generation Script

Write a Playwright script that opens the VTC GenAI Portal, navigates to Image Generation, selects the NANO BANANA model, enters a prompt describing a futuristic city skyline, sets the aspect ratio to 16:9, clicks Generate, waits for the image to appear, and downloads it to ./generated-images/.

▶ Batch Image Generation

Write a Playwright script that reads a list of image descriptions from a JSON file and generates one image for each description from the VTC GenAI Portal using DALL-E 3. Save each image with a filename based on its index and wait 2 minutes between each generation to allow time for processing.

▶ Video Generation Script

Write a Playwright script that opens the VTC GenAI Portal, navigates to Video Generation, enters a text prompt describing a sunrise over a mountain, sets the duration to 5 seconds and resolution to 720p, clicks Generate, waits for the video to be ready, and downloads the MP4 file to ./generated-videos/.
